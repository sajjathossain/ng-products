## Getting Started

### Prerequisites

- Node
- NPM

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:sajjathossain/ng-products.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

- Run your local dev server
  ```sh
  npm start
  ```
